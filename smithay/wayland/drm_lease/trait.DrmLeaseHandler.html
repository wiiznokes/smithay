fn lease_request( &mut self, node: DrmNode, request: DrmLeaseRequest ) -> Result<DrmLeaseBuilder, LeaseRejected>

A client has issued a new request.

The request only contains connectors and the compositor is lease any resources it may seem fit. It is recommended however to lease the requested connectors and any resources necessary to drive them (free CRTCs per connector and at least their primary plane) or reject the request.

To reject the request return Err(LeaseRejected::default()), otherwise return a DrmLeaseBuilder with all the resources added.
