fn accept(&mut self, mime_type: Option<String>, seat: Seat<Self>)

The client can accept the given mime type.

If mime_type is None, the client cannot accept any of the offered mime types. If the last accepted mime_type is None, the drag-and-drop operation will be cancelled. That is kept track of internally, this callback is informational only.

  • mime_type - The accepted mime type
  • seat - The seat on which the DnD action was chosen.
